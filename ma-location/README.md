# Hackathon 2025 : M.A Location

## Développement

Frameworks : **React** + **ViteJs**

Une fois le repo cloné, il faut :
- Aller dans le dossier **"ma-location"** : ` cd ./ma-location `
- Télécharger les dépendences : `npm i`
- Et pour lancer le serveur : `npm run dev`
